# Sky-Eye-Inspection-and-Operation-Intelligent-Video-Recognition-Technology-for-Drone-Patrols
聚焦人工智能产业在智慧物流领域的快速发展，本项目针对传统货运车辆信息识别依赖人工、效率低下、风险高以及货物状态实时监测困难等行业急需解决的问题，拟利用无人机巡航获取的高清视频流，应用先进的AI视觉识别技术，实现货运车辆车号的自动识别与货物状态的智能监测。通过非接触式的视频采集与计算分析，解决人工核查效率低、安全隐患大、成本高昂等痛点，旨在提升物流运输环节的自动化水平和监管效率。项目预期通过技术创新，显著提高物流作业效率与安全性，降低运营成本，推动智慧物流体系的建设，并计划交付相对应的算法模型和对应的分析展示工具。
